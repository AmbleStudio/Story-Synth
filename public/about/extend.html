How to Extend Story Synth

Story Synth is designed to be adapted to new types of storytelling games with novel mechanics. The framework is open sourced under an MIT License and you’re free to do almost anything you want with it. Check out the Story Synth codebase on GitHub.

Whether you're doing a small extension of the app by tweaking the sheet template or attempting a more drastic hack, I recommend you get familiar with the Vue.js documentation. Vue.js is incredibly powerful and fairly quick to use, once you’re up to speed.

The code for the homepage and room selector live in /components/RoomPicker.vue – if you’re in a game room, then the RoomPicker just shows the current URL and number of players on the page.

The main code for each game lives in that template’s .vue file in the /component folder.

As Story Synth progresses, I hope to add much more to this section. Until then, feel free to reach out with any questions: randy@storysynth.com
