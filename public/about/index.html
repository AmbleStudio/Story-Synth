About Story Synth

Overview

Story Synth is a platform for developing, playtesting, and publishing lightweight storytelling games on the web. The mission is to lower the barrier to playtesting games online and encourage a flourishing of new designs.

There are a few key features:

  • Designers can create their game in an Google Spreadsheet and can playtest by pasting the link into Story Synth – no coding knowledge needed.
  • Story Synth games take place in “Rooms” where content is synchronized for everyone in the same Room. In other words, everybody at the same URL automatically sees the same content.
  • There are multiple templates for different types of games and, if you’re comfortable coding, you can add your own games

You can see the templates in action (with placeholder content) at storysynth.org.

Visit Dawn of the Monster Invasion for example of a published game that runs on Story Synth.

Check out the repository on GitHub to explore and download the codebase.

Use cases and guides

Story Synth is for game designers with one of three use cases:

  1. Playtesting a game that uses one of the existing templates (no coding needed)
  2. Publishing a game that uses one of the existing templates, on their own website, so that others can easily play it (some, simple coding needed)
  3. Extending one one of the templates or creating their own template, to add mechanics that suit their game (serious coding needed)

Here are guides for each of the use cases:

Available templates

Story Synth currently has four different game templates you can use. Here’s an overview of each template, its influences, and a link to the template spreadsheet.

Shuffled

Shuffled Template Spreadsheet

This template is inspired by Alex Robert’s game For the Queen (buy) (SRD).

Games following this template have:

  • A series of instructions, spread over short cards that appear in a fixed order
  • A number of storytelling prompts that are shuffled in order
  • A final card that brings the game to a close, which is always the last card
  • An x-card for safety

Unlike For the Queen, there are no Queen cards to choose from and the final card is always last, rather than shuffled into the deck.

The broader format of ordered instructions followed by random prompts lends itself to a wide variety of games that stray far from For the Queen.

Timed

Timed Template Spreadsheet

This template is inspired by Jason Morningstar’s game Find/Fix/Finish (buy).

This game has a shared timer and private prompts are sent to players at specific times. Players can pause, unpause, and restart the game at any point.

Unlike Find/Fix/Finish, there aren’t any decision points where a player can choose from different options and which result in different content later. There also isn’t space for links to reference material, though you could include them in the prompts.

Secrets

Secrets Template Spreadsheet

This template is similar to the Timed template but prompts are tied to each turn instead of a timer and turns are manually advanced.

In each turn, each player will receive one public prompt (seen by all) and one private prompt (seen only by them).

Monster

Monster Template Spreadsheet

This template is inspired by Randy Lubin’s game Dawn of the Monster Invasion (play online).

This template has:

  • A series of ordered instructions at the beginning of the game
  • A series of rounds, with each round containing several prompts to choose from
  • The prompts are fixed to the specific rounds (i.e. not random) and players will face the same options each playthrough

Technology used

Story Synth uses:

  • Vue.js for the frontend framework
  • Firebase for syncing game state, presence, and website hosting
  • VueFire to sync Firebase and Vue
  • Axios to fetch data from Google Sheets
  • Bootstrap for the design framework

This means Story Synth is serverless and completely free at moderate usage.
